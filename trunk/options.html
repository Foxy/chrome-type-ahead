How to use this extension

  • Write directly or press / to search text.
  • Press ' to search text links.
  • Use F3 or Control+G or Alt+N/P to switch between matches.
  • Use F4 to toggle matching mode (text/links) once the search has started.

Issue 150

Although this extension tries to do the job the best it can, by no means it can work as well and fast as a core feature. Issue 150 was opened a long time asking for type-ahead to be implemented in Chromium, but -to everybody's surprise- the report was closed the same day with a WontFix. Dozens of angry comments didn't change that, so all we can do now is star the issue. If the issue makes its place amongst the most voted request maybe Chromium developers will rethink their decision.

Check the Chrome-ahead-find website for more information about this extension.
